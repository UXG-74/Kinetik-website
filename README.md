# Kinetik website

Marketing site for **KINETIK** — React, TypeScript, and Vite. Layout keyed to the Figma homepage (`2108:810`) where noted in source.

**Path:** `~/Kinetik/website` (sibling of **Kinetik App** in `~/Kinetik/app`).

## Scripts

| Command            | Purpose                 |
| ------------------ | ----------------------- |
| `npm run dev`      | Local dev server (Vite) |
| `npm run build`    | Production build        |
| `npm run preview`  | Preview built output      |
| `npm run lint`     | ESLint                  |

## Git remote

Configured for `git@github.com:UXG-74/kinetik-website.git`. Create an empty **`kinetik-website`** repository under UXG‑74 before the first push, then:

```bash
git push -u origin main
```
